Acids, Bases and Salts

How do Acids and Bases React with Metals?

Acid + Metal → Salt + Hydrogen gas
2NaOH + Zn → Na₂ZnO₂ + H₂

How do Metal Carbonates and Metal Hydrogencarbonates React with Acids?

Metal carbonate/Metal hydrogencarbonate + Acid → Salt + Carbon dioxide + Water
Na₂CO₃(s) + HCl(aq) → NaCl(aq) H₂O(l) + CO₂(g)
NaHCO₃(s) + HCl(aq) → NaCl(aq) H₂O(l) + CO₂(g)
Ca(OH)₂(aq) + CO₂(g) → CaCO₃(s) + H₂O(l)
CaCO₃(s) + H₂O(l) + CO₂(g) → Ca(HCO₃)₂(s)

How Acid & Base React with each other?

Base + Acid → Salt + Water
NaOH(aq) + HCl(aq) → NaCl(aq) + H₂O(l)

Reaction of Metallic Oxides with Acids

Metal oxide + Acid → Salt + Water

When Acid or a Base is added to Water Solution?

HCl + H₂O → H₃O⁺ + Cl⁻
H⁺ + H₂O → H₃O⁺
NaOH(s) → Na⁺(aq) + OH⁻(aq)
KOH(s) → K⁺(aq) + OH⁻(aq)
Mg(OH)₂(s) → Mg²⁺(aq) + 2OH⁻(aq)

Sodium hydroxide

2NaCl(aq) + 2H₂O(l) → 2NaOH(aq) + Cl₂(g) + H₂(g)

Bleaching powder

Ca(OH)₂ + Cl₂ → CaOCl₂ + H₂O

Baking soda

NaCl + H₂O + CO₂ + NH₃ → NH₄Cl + NaHCO₃
2NaHCO₃ → Na₂CO₃ + H₂O + CO₂
NaHCO₃ + H⁺ → CO₂ + H₂O + Sodium salt of acid

Washing soda

Na₂CO₃ + 10H₂O → Na₂CO₃.10H₂O

Plaster of Paris

CaSO₄.(1/2)H₂O + 1(1/2)H₂O → CaSO₄.2H₂O
