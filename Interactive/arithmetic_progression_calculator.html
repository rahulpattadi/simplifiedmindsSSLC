About Arithmetic Progressions

An arithmetic progression (AP) is a sequence where the difference between each consecutive term is constant. This constant difference is called the common difference (d).

  • • Terms are denoted as: a₁, a₂, a₃, ..., aₙ
  • • Common difference: d = a₂ - a₁ = a₃ - a₂ = ... = aₙ - aₙ₋₁
  • • The sequence: a, a+d, a+2d, a+3d, ..., a+(n-1)d
